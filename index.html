ML Architecture

Linear Regression
Task: Regression

Predicts exact energy consumption (kWh) based on all household features. Optimized with StandardScaler for fast convergence.

R² · MAE · RMSE
Decision Tree
Task: Binary Classification

Classifies households as High Usage vs Normal. Interpretable splits reveal which features drive excessive consumption.

Accuracy · Precision · Recall · F1
KNN Classifier
Task: Multi-class Classification

Predicts Efficiency Category (Low/Medium/High) using distance-weighted k=7 neighbors on normalized feature space.

Accuracy · Precision · Recall · F1
K-Means
Task: Clustering

Segments households into 4 consumption clusters — from Eco Savers to Energy Intensive — enabling targeted interventions.
